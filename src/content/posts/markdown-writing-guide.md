## 标题层级要清晰

合理使用 `##` / `###`，既方便读者快速浏览，也便于生成文章目录（TOC）。

## 代码块记得标注语言

标注语言后，渲染器才能实现语法高亮：

```js
console.log('Hello, Markdown!');
```

## 用表格呈现对比

| 语法 | 用途 |
| --- | --- |
| `#` | 一级标题 |
| `>` | 引用 |
| `-` | 列表项 |

## 引用用于强调

> 好的技术文章，重点不在炫技，而在把一件事讲清楚。

## 链接与图片

内联链接写法是 `[文本](URL)`，图片则是 `![替代文本](URL)`。

## 小结

Markdown 的精髓是「专注内容、忽略排版」。掌握这套轻量语法，你就能把精力放在真正重要的表达上。
